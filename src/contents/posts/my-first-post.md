## 大家好，我是飯魚

大家好，我是飯魚。

目前是一名資訊相關科系的大學生，平常主要接觸 Web 開發、程式設計以及一些自己感興趣的專案。

從開始接觸寫程式以來，我一直覺得「做出一個實際能使用的東西」是一件很有成就感的事情。

不管是一個小工具、一個遊戲，還是一個網站，只要最後能看到自己的想法變成真正可以運作的作品，都會讓我覺得學習程式是一件很有趣的事情。

---

## 為什麼想建立這個 Blog？

其實以前學習程式的過程中，常常會遇到一些問題。

可能是一個語法搞不懂、一個 Bug 找很久，或是一個功能研究了很多資料才成功完成。

很多時候解決問題後，過一陣子又會忘記當初是怎麼處理的。

所以我想建立這個 Blog，把自己學習與開發過程中的紀錄整理下來。

除了方便未來回頭查看，也希望這些內容能幫助到剛好遇到相同問題的人。

---

## 這個網站是怎麼做的？

這個網站主要使用 Next.js 建立。

目前使用的技術包含：

- Next.js
- React
- TypeScript
- Tailwind CSS

文章內容則是使用 Markdown 撰寫，透過靜態生成的方式建立頁面。

這樣的好處是不需要額外架設資料庫，就能擁有快速的載入速度以及良好的 SEO 表現。

---

## 這裡會分享什麼？

這個 Blog 主要用來記錄自己的學習與開發過程。

內容可能包含：

- 開發專案時遇到的問題與解決方式
- 學習新技術時的整理與心得
- 一些有趣的想法與實驗
- 製作作品時的過程紀錄
- 平時累積下來的各種筆記

不一定每篇文章都是完整的教學，有些可能只是紀錄自己探索過程中的想法與經驗。

我希望這裡更像是一個開發筆記，而不是單純分享答案的地方。

---

## 給未來的自己

開始寫這篇文章時，還有很多東西正在學習。

有些技術可能只是剛接觸，有些專案也還有很多不足的地方。

不過我覺得學習程式最有趣的地方，就是可以不斷發現新的問題，然後慢慢找到解決方法。

希望之後回來看這個 Blog 時，可以看到自己一路學習、嘗試以及累積的過程。
